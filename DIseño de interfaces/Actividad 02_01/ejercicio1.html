¿En qué año se creó CSS y quién fue su creador?

CSS (Cascading Style Sheets) se creó en 1996. El principal creador de CSS fue Håkon Wium Lie. Él propuso la idea en 1994 mientras trabajaba en el CERN y luego colaboró con otros para desarrollarlo formalmente.

¿Por qué surgió la necesidad de separar el contenido HTML del diseño CSS?

La necesidad de separar el contenido HTML del diseño CSS surgió para:

- Mejorar la organización

- Aplicar un mismo estilo a múltiples páginas

- Cambiar el diseño sin alterar el contenido

- Mejora la compatibilidad con distintos dispositivos y tecnologías

¿Qué mejoras trajo consigo CSS3 respecto a versiones anteriores?

CSS3 trajo varias mejoras importantes, entre ellas:

- una selección más precisa de elementos

- Animaciones y transiciones

- Soporte para diseño responsivo

- Nuevas propiedades de diseño como bordes redondeados, sombras y gradientes

- Flexbox y Grid para layouts más flexibles y complejos

¿Qué ventajas ofrece CSS3 para diseñar interfaces modernas?

- Interactividad y dinamismo

- Diseños responsivos

- Estilos visuales avanzados

- Layouts flexibles con Flexbox y Grid
